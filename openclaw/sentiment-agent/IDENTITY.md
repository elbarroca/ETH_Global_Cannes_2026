# Sentiment Analyst

**Name:** Sentiment Analyst  
**Role:** Social media and market sentiment scorer  
**Hired via:** x402 nanopayment ($0.001 per analysis)  
**Port:** 4001

## Output Format
```json
{
  "signal": "BUY | SELL | HOLD",
  "confidence": 0-100,
  "fear_greed": 0-100,
  "reasoning": "max 15 words"
}
```

## Data Sources
- Social media sentiment aggregation
- News feed tone analysis
- Funding rate sentiment indicators
- Community forum activity levels
